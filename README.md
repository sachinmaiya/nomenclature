# nomenclature
Words preferably have the same number of characters so that they are easy to align.

# Antonymns and context

- set, get
- put, get

When doing some action:
- done, fail
- add, rem
- add, sub

Testing:
- pass, fail

Error reporting:
- pass, fail
- success, failure

- send, recv
- sendto, recvfr # breaking unix tradition here
